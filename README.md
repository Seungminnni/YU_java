# YU Java 프로젝트

이 프로젝트는 Java 프로그래밍 학습을 위한 종합 예제 코드들을 포함하고 있습니다.

## 프로젝트 구조

```
YU_java/
├── Practice_01/ (11개 파일)
│   ├── Hello.java           # 기본 프로그램 구조
│   ├── Example01.java       # 상수와 변수
│   ├── Datatype01.java      # 정수형 데이터 타입
│   ├── Datatype02.java      # 실수형 데이터 타입
│   ├── Datatype03.java      # 문자형 데이터 타입
│   ├── Datatype04.java      # 문자열 데이터 타입
│   ├── InputOutput01.java   # 기본 입출력
│   ├── InputOutput02.java   # 이스케이프 시퀀스
│   ├── Operator01.java      # 산술 연산자
│   ├── Operator02.java      # 비교 연산자
│   └── Welcome.java         # 메뉴 시스템
├── Homework_01/ (5개 파일)
│   ├── hw1.java            # 데이터 타입 종합
│   ├── hw2.java            # printf 실습
│   ├── hw3.java            # Scanner 실습
│   ├── hw4.java            # 변수 교환
│   └── hw5.java            # 시간 변환
└── README.md
```

## 파일 설명

### Practice_01 패키지 (기본 실습 - 11개 파일)

#### Hello.java
- **목적**: Java 프로그램의 가장 기본적인 구조를 보여주는 예제
- **내용**: 간단한 변수 선언과 System.out.println()을 사용한 출력
- **학습 포인트**: main 메소드 구조, 변수 선언, 기본 출력
- **실행 결과**: "Hello World" 스타일의 기본 메시지 출력

#### Example01.java
- **목적**: Java의 상수와 변수 개념의 차이를 이해하기 위한 예제
- **내용**: final 키워드를 사용한 상수 선언과 일반 변수의 차이점 시연
- **학습 포인트**: final 키워드 사용법, 상수 vs 변수의 특징
- **실행 결과**: 상수 SPEED와 변수 count의 값 변화 비교 출력

#### Datatype01.java
- **목적**: Java의 기본 정수형 데이터 타입들을 종합적으로 학습
- **내용**: int, short, byte, long 데이터 타입의 선언과 값 할당
- **학습 포인트**: 각 데이터 타입의 메모리 크기와 범위 이해
- **실행 결과**: 각 데이터 타입의 값들을 순서대로 출력

#### Datatype02.java
- **목적**: Java의 실수형 데이터 타입들을 학습
- **내용**: float와 double 데이터 타입의 선언과 값 할당
- **학습 포인트**: 실수형의 정밀도 차이, f/F 접미사 사용
- **실행 결과**: 실수 값들의 출력과 정밀도 확인

#### Datatype03.java
- **목적**: Java의 문자형 데이터 타입을 학습
- **내용**: char 데이터 타입을 사용한 문자들의 선언과 출력
- **학습 포인트**: 작은따옴표 사용, 유니코드 지원
- **실행 결과**: 개별 문자들이 순서대로 출력

#### Datatype04.java
- **목적**: Java의 문자열 데이터 타입을 학습
- **내용**: String 클래스를 사용한 문자열 선언과 조작
- **학습 포인트**: 큰따옴표 사용, 문자열 연결 연산자(+)
- **실행 결과**: 두 문자열의 개별 출력과 연결된 결과 출력

#### InputOutput01.java
- **목적**: Java의 기본 입출력 기능을 학습
- **내용**: System.out.println()을 사용한 한글 출력
- **학습 포인트**: 한글 지원, println() 메소드의 특징
- **실행 결과**: 이름과 나이 정보의 깔끔한 출력

#### InputOutput02.java
- **목적**: Java의 이스케이프 시퀀스를 학습
- **내용**: \t(탭)과 \n(줄바꿈)을 사용한 서식 있는 출력
- **학습 포인트**: 이스케이프 시퀀스의 종류와 사용법
- **실행 결과**: 탭과 줄바꿈이 적용된 정렬된 출력

#### Operator01.java
- **목적**: Java의 산술 연산자들을 종합적으로 학습
- **내용**: +, -, *, /, % 연산자를 사용한 계산
- **학습 포인트**: 각 연산자의 기능, 자동 형 변환
- **실행 결과**: 다양한 산술 연산의 결과 값들 출력

#### Operator02.java
- **목적**: Java의 비교 연산자들을 학습
- **내용**: ==, !=, <, >, <=, >= 연산자를 사용한 비교
- **학습 포인트**: 비교 연산자의 결과 타입(boolean)
- **실행 결과**: 각 비교 연산의 true/false 결과 출력

#### Welcome.java
- **목적**: 실제 응용 프로그램의 메뉴 시스템 구현
- **내용**: 탭을 사용한 깔끔한 메뉴 정렬과 디자인
- **학습 포인트**: 실전적인 출력 formatting, UI 디자인
- **실행 결과**: 쇼핑몰 스타일의 메뉴 화면 출력

### Homework_01 패키지 (숙제 실습 - 5개 파일)

#### hw1.java
- **목적**: Java의 모든 기본 데이터 타입을 종합적으로 실습
- **내용**: String, int, float, char, boolean 데이터 타입 모두 사용
- **학습 포인트**: 데이터 타입의 종합적 이해와 활용
- **실행 결과**: 개인 정보의 종합적인 출력

#### hw2.java
- **목적**: printf 메소드를 사용한 고급 출력 formatting 학습
- **내용**: %.2f 포맷을 사용한 소수점 둘째자리 고정 출력
- **학습 포인트**: printf의 포맷 지정자 사용법
- **실행 결과**: 사각형 면적과 둘레의 정밀한 출력

#### hw3.java
- **목적**: Scanner 클래스를 사용한 사용자 입력 처리 학습
- **내용**: 섭씨 온도 입력받아 화씨로 변환
- **학습 포인트**: Scanner 클래스 사용법, 실수 입력 처리
- **실행 결과**: 입력된 섭씨와 변환된 화씨 동시 출력

#### hw4.java
- **목적**: 변수 값 교환 알고리즘 구현
- **내용**: 임시 변수를 사용한 두 변수 값 교환
- **학습 포인트**: 알고리즘 구현 능력, 임시 변수 활용
- **실행 결과**: 교환 전후 값들의 비교 출력

#### hw5.java
- **목적**: 시간 변환 알고리즘 구현
- **내용**: 총 초를 시:분:초 형식으로 변환
- **학습 포인트**: 수학적 계산 알고리즘, printf 0패딩
- **실행 결과**: HH:MM:SS 형식의 시간 출력

## 학습 내용 체계

### 📚 **핵심 개념**
- **기본 문법**: 클래스, 메소드, 변수 선언
- **데이터 타입**: 6가지 기본 타입 + String
- **입출력**: println, print, printf, Scanner
- **연산자**: 산술, 비교, 대입
- **제어 구조**: 조건문, 반복문 기초

### 🛠️ **실전 기술**
- **형식화된 출력**: printf와 포맷 지정자
- **사용자 입력**: Scanner 클래스 활용
- **알고리즘 구현**: 변수 교환, 시간 변환
- **메모리 관리**: 기본 데이터 타입의 특징
